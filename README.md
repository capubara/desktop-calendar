# Fantasia Calendar

Учебное desktop-приложение календаря в стиле Fantastical: PHP + SQLite + адаптивный интерфейс + Electron-обертка для запуска как самостоятельное .exe.

## Главное

- PHP является основной серверной частью и работает с базой данных SQLite.
- База создается автоматически в storage/app.sqlite.
- Есть регистрация, авторизация через пароль и одноразовый код из письма, запоминание устройства.
- Первый зарегистрированный пользователь автоматически получает роль администратора.
- Есть личный кабинет: темы, цвет календаря, устройства, безопасность, 2FA, язык, иконка, помощь.
- Есть календарь, Today-режим, отметка выполненных дел и приглашения для совместного доступа.
- Интерфейс адаптирован под desktop и мобильную ширину.
- Electron-обертка запускает PHP-сервер внутри приложения и открывает окно.

## Запуск как веб-приложение

php -S 127.0.0.1:8080 -t public

Открыть: http://127.0.0.1:8080

## Демо-почта

Коды подтверждения записываются в storage/mailbox.log. Для учебной проверки код также показывается в интерфейсе после отправки.

## Desktop и .exe

В папке desktop описана Electron-обертка. Для финальной сборки PHP можно положить в desktop/vendor/php/php.exe, тогда .exe будет запускать сервер самостоятельно.

cd desktop
npm install
npm run start

Сборка:

npm run dist

Результат появится в desktop/dist.
